Traductor

Código del Ejemplo

traductor.py 
from googletrans import Translator

# Crear el objeto traductor
translator = Translator()

# Diccionario de idiomas disponibles
idiomas = {
    '1': ('en', 'Inglés'),
    '2': ('pt', 'Portugués'),
    '3': ('de', 'Alemán'),
    '4': ('fr', 'Francés'),
    '5': ('it', 'Italiano'),
    '6': ('ja', 'Japonés'),
    '7': ('zh-cn', 'Chino'),
    '8': ('ko', 'Coreano')
}

# Solicitar el texto a traducir
texto = input("Escribe el texto a traducir: ")

# Mostrar opciones
print("\n=== IDIOMAS DISPONIBLES ===")
for num, (codigo, nombre) in idiomas.items():
    print(f"{num}. {nombre}")

# Pedir que elija 3 idiomas
print("\nElige 3 idiomas (separa los números con espacios, ej: 1 2 3)")
seleccion = input("Tu elección: ").split()

# Traducir a los idiomas seleccionados
print(f"\nOriginal: {texto}\n")
for num in seleccion[:3]:  # Solo toma los primeros 3
    if num in idiomas:
        codigo, nombre = idiomas[num]
        traduccion = translator.translate(texto, dest=codigo).text
        print(f"{nombre}: {traduccion}")
    else:
        print(f"Opción '{num}' no válida")

Glosario de Funciones

translator
Es el nombre de la clase utilizada googletranspara realizar la traducción.
for
Se utiliza para iterar sobre una secuencia de elementos (como una lista, tupla, cadena o diccionario) y ejecutar un bloque de código por cada elemento.
in
Es un operador de pertenencias que comprueba si un valor existe dentro de una secuencia(como una lista, tupla, cadena o diccionario) y devuelve True si lo encuentra, o False si no.
